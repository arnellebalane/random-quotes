# random-quotes

Get random quotes.

## Installation

```bash
$ npm install --save random-quotes
```

## Usage

```js
const randomQuotes = require('random-quotes');

randomQuote();
//  {
//    body: 'It's not poisonous...',
//    author: 'Famous Last Words'
//  }
```

## API

- **`randomQuote()`**
  - returns a random quote object with the following keys: `author`, `body`
